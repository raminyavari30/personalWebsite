Portfolio

Caffiend is a webapp that allows users to find their new favorite menu item at a given cafe. Users can search for cafes, view menus sorted by item rating, and add their own cafes and menu items.

Caffiend is built in Angular on the front end, with a Node and Express backend that communicates with an SQLite3 database.

CityDash is a simple multitool application that allows users to select specific city related informaton to display on their own customizable personal dashboard.

CityDash is built in React on the front end, with Node and Express on the backend. Firebase stores all the user's data of preferred layouts and other user information.

Itinderary is a web application that generates activities to do based on how the user is feeling. Itinderary cleverly uses the Yelp API combined with its own searching algorithm to return accurate results based on the mood of the user.

Itinderary is built in React, Redux, and React Router on the front end, with Node and Express on the back end.
